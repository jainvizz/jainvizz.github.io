Introduction

Sass stands for "Syntactically Awesome Style Sheets".


Sass is a scripting language (Preprocessor) that is interpreted into Cascading Style Sheets (CSS).


Sass is the most mature, stable, and powerful professional grade CSS extension language in the world.

Nesting your Sass

◈ Nesting allows you to write selectors that mimic the structure of your HTML.

◈ This allows you to use shortcuts to create your CSS.

◈ Nesting can extend as many levels deep as you wish.

Using Mixins

◈ Mixins are like functions.

◈ Mixins allow you to define styles that can be re-used throughout the stylesheet.

◈ They can even take arguments which allows you to produce a wide variety of styles.

Selector Inheritance

◈ Provides a simple way to allow a selector to inherit the styles of another one

◈ One class can have all the styles of another class, as well as its own specific styles

@import & Parent Reference

◈ A partial is a file that contains some of your Sass. We use partials to organize our work.

◈ In Sass, & references the parent selector of a declaration block, allowing the selector to be referenced from within itself.

Using Lists and @each

The @each directive takes the form @each $var in <list>.

            $list: (red, pink, black, yellow, blue);

            @each $author in $list{
              .photo-#{$author} {
                color: #{$author};
              }
            }
If else conditions

The @if control directive takes a SassScript expression and processes its block of styles if the expression returns anything other than false.


                @if $i == 1 {
                     /* ..... */           
                }@else if $i > 1 and $i < 6 {
                    /* ..... */ 
                }@else if $i == 6{
                    /* ..... */
                }

For Loops

The @for directive comes in two forms.

1. The first option is @for $var from <start> through <end>

2. The second option is @for $var from <start> to <end>

While Loops 

                $types: 4;
                $type-width: 20px;

                @while $types > 0{
                  .div-#{$types}
                  {
                  width: $type-width + $types
                  }
                  $types: $types - 1
                }
